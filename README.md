# TribeTrack

A social habit tracking system that builds accountability through group streaks, proof-based completion, and recovery mechanisms.

## Features
- Group habit tracking
- Streak system
- Smart nudges
- Recovery system

##  Tech Stack
- Python (Flask)
- HTML/CSS

##  How to Run
1. Clone the repo
2. Run `python app.py`
3. Open http://127.0.0.1:5000
